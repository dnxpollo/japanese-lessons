# Nihon no Nichijō 🇯🇵

Plataforma interativa para aprender japonês do cotidiano, desenvolvida com **React + TypeScript**.

## ✨ Sobre o projeto

"Nihon no Nichijō" transforma elementos do dia a dia japonês em uma experiência de aprendizado visual, prática e divertida. A proposta é estudar a língua japonesa enquanto pratico desenvolvimento com React — e quem sabe ajudar outras pessoas com o mesmo interesse.

## 📚 Conteúdo disponível

- **Hiragana** — Alfabeto fonético básico
- **Katakana** — Palavras estrangeiras e nomes científicos
- **Kanji** — Ideogramas com significados complexos
- **Quiz de Cultura Japonesa** — Teste seus conhecimentos!

## 🛠️ Tecnologias usadas

- React
- TypeScript
- CSS 
- Git + GitHub
